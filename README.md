# OTUS: React.js Developer, домашние задания Плющенков Дмитрий

## Консольный калькулятор:
Используемая конфигурация:
* Babel
* Webpack
* Typescript
* Eslint
* Jest
* Prepush hooks / husky

## Запуск в консоли

`npm run calc`

Возможные операции: * / + - ( )

## Запуск тестов

`npm run test`
